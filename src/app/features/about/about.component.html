About this quiz

  • There are 10 random questions (per request) taken from https://the-trivia-api.com/.
  • Each question has only one right answer.
  • Each right answer is valid only one point.
  • Once the answer is given, the options get disabled, therefore there is no possibility in changing it.
  • There is a possibility to check the correct answer, after the answer is given.
  • There is a possibility to check all right answers by going to "Quiz Answers" on the navigation, however you need admin priviledges for that.
Note: Please do not refresh the page, the ngrx store is not persistent therefore all your data will get lost!
